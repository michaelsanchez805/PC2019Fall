Fall Course Information

Dates and Prices

The fall courses are broken into 7 sessions, two hours each on Sunday mornings, from:

September September 22nd to November 17th (Skipping November 3rd and 10th)

Each week you can pick which time slot to go to 9:00am-10:30am or 11:00am-12:30pm.

We will be flexible and creative with make-ups if your child has scheduling challenges.

  • The fall course fee is $200 for the 7 weeks, which includes a t-shirt, Rubix cube, laptop stickers and other fun prizes along the way.
  • All students MUST bring a computer and charger to camp each day to code with - any computer that can connect to the internet will work.
  • If you would like, you can sign up for two different tracks and attend both time slots for a discounted fee of $360

    • Available Tracks This Summer

    Select one of the following topics as your track for the:

    Web Development - Build your own websites with HTML, CSS, and JavaScript in this fun course that's great for beginners.

    Game Development - Use JavaScript and p5.js to create fun and popular games like Pong and Flappy Bird, while learning essential programming concepts along the way including variables, conditional statements, and loops. Beginners welcome.

    Artificial Intelligence - A course for the Cool Coding veteran, or for interested students who want to move beyond the basics of coding. This course will delve into machine learning and how you can program a computer to think for itself.

    Little Coders (Grades 1-3) - Get exposure to coding concepts at an early age with this game-filled course that will keep everyone laughing. Students must know how to read.

    The Power Coding Method

    It is important to us to make sure that our coding lessons are engaging for everyone. We do this by incorporating fun games to teach concepts, and coding amazing projects to show your friends and family. Get ready to solve puzzles, win prizes, and have the adventure of a lifetime!

    Each class will always have a lot of instructors in the room, ensuring that everyone gets the attention they need and are able to create and learn an endless amount of knowledge.
